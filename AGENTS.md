# student_system_pro

基于 FastAPI + MySQL 的多用户学生管理系统。支持注册登录、数据隔离、搜索分页、删除确认等完整企业级 CRUD。

## 技术栈
- 后端：FastAPI, Uvicorn, PyMySQL
- 数据库：MySQL
- 前端：原生 HTML5/CSS3/JavaScript (ES6+)
- 部署：cpolar 内网穿透

## 运行方式
`ash
uvicorn main:app --reload
`
访问 http://127.0.0.1:8000
